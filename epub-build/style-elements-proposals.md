## 4. Drop Caps (Initialen)

Erster Buchstabe jedes Kapitels als 3-Zeilen-Initiale. Funktioniert nur im EPUB (CSS `::first-letter`).

→ Wird automatisch für alle 16 Kapitel + Vorwort + Epilog gesetzt.

## 5. Szenische Trenner (Ornamental Breaks)

Zwischen thematischen Abschnitten innerhalb eines Kapitels — statt Leerzeile ein Ornament: **⁂** oder **✦** oder eine Zierlinie **───── ◆ ─────**

Empfehlung: `✦` — modern, dezent, passt zum Tech/Investment-Thema.

→ Wird an jeder Stelle gesetzt, wo aktuell eine H3-Subheading steht und ein deutlicher Themenwechsel stattfindet (nicht bei jeder H3, nur bei echten Brüchen).
